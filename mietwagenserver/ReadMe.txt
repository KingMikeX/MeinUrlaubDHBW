_______Installationsbeschreibung:______ !!!!( Müsst ihr nicht machen )

npm i express
npm i mongoose
npm i cors
npm i dotenv
npm i node-fetch
npm i -g nodemon

Automatische Reload zu nutzen und die App zu starten:
  -> nodemon server.js (die Applicationen)

_____Git-Einbindung:_______

Bevor ihr irgendwelche veränderungen macht und die Sachen versucht auf Git zu pushen, erstellt erstmal wie im Praxissem bei den Grasshopper ein neuen Branch.

Wie macht ihr das?:
  -> git checkout -b orgin feature/"IrgendEinName"
  -> anschließende erstelltung eines remote repositories
  -> des weiteren müsst ihr auf dann den Branch mit einem Local repo verknüpfen
  -> anschließend könnt ihr programmieren wie ihr wollt ohne den Haupt in Github kaput zu machen

  Als letztes um alles zu starten und zu installieren einfach npm install machen und alle node_modules die in package.json drin sind werden
